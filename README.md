## 🎮 프로젝트 개요

- **장르**: 모바일 라인 디펜스 (3레인 오토배틀 + 타이밍 스킬)
- **컨셉**: 선사시대부터 메카닉 미래까지, 전통(승복·삼베·한옥)과 하이테크(레이저·네온 HUD·사이버펑크)가 융합된 시대 방어전
- **플랫폼**: 모바일, 가로 화면 최적화
- **현재 상태**: 테스트 버전(MVP) 기획 중

## 🧩 핵심 시스템

1. **조작**: 영웅 4명이 우측 성벽에 고정, 자동 공격 + 유저 타이밍 스킬 터치
2. **레인**: 상/중/하 3레인, 캐릭터별로 완전히 다른 대응 방식
   - 🧝‍♀️ 엘프 저격수 — 레인 무관 점사
   - 🐯 수인 조종사 — 맵 전체 자유 이동 광역 폭격
   - 🌾 노비 폭탄 장인 — 레인 지정 CC
   - 📿 스님 장군 — 레인 관통 레이저
3. **자원**: 입장 전 파티 세팅(영구 성장) + 인게임 실시간 골드 오버클럭(휘발성 강화)

## 🧪 테스트 버전(MVP) 범위

- **포함**: 1~4단계 (4캐릭터 풀 로스터 완성까지)
- **제외/보류**: 5단계 이후, BM 구조, 로비 소셜 펀 콘텐츠
