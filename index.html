Overview

시대를 건너는 4명, 하나의 성벽

귀여운 공룡부터 거대 총기 로봇까지, 몬스터와 영웅 모두 "전통 소재 + 사이보그화"라는 하나의 룰로 진화한다. 유저는 우측 성벽에 고정된 영웅들의 오토 전투를 지켜보다, 결정적 순간에 스킬을 터치해 3개 레인의 위기를 관리한다. 승복 입은 승병장이 레이저 캐논을 쏘고, 노비가 빙결탄을 던지는 이 부조화가 게임의 정체성이다.

01 — 조작 방식

오토 전투 + 타이밍 스킬

영웅은 우측 성벽에 고정, 자동 공격. 유저는 타이밍에 맞춰 스킬 버튼만 터치한다.

02 — 레인 구조

상 · 중 · 하, 4가지 다른 해법

엘프는 레인 무관 점사, 수인은 맵 전체 자유 폭격, 노비는 레인 지정 CC, 스님은 레인 관통 — 같은 위협에 4개의 다른 답이 존재한다.

03 — 자원 시스템

프리셋 파티 + 실시간 오버클럭

입장 전 파티는 영구 성장, 인게임 골드는 그 판 한정 스킬 강화 — 이중 루프로 매 플레이의 선택을 다르게 만든다.

Test Version Scope

이번 빌드가 증명해야 할 것

"4명이 모였을 때, 레인 판단 + 스킬 타이밍 + 골드 배분이 유의미한 선택으로 느껴지는가."

Included — MVP

  • 1~4단계 풀 플레이 (4캐릭터 로스터 완성)
  • 3레인 조작 + 오토배틀 코어 루프
  • 실시간 오버클럭 성장 시스템
  • 스테이지별 웨이브 밸런스 초안

Deferred — Post-Test

  • 5단계 이후 (전 캐릭터 합류~최종보스)
  • BM 구조 (가챠 · 스킨 판매)
  • 로비 소셜 펀 콘텐츠
  • 기가-아스널 보스 페이즈 기믹

Playtest Checklist

  1. 레인 판단이 직관적으로 읽히는가 (위험 레인이 즉시 파악되는가)
  2. 4개 스킬의 쿨타임·사거리가 겹치지 않고 각자 존재 이유가 있는가
  3. 오버클럭 골드 분배가 고민되는 선택인가, 항상 정답이 정해져 있는가
  4. 캐릭터가 늘어날 때(2→3→4단계) 난이도 체감이 자연스럽게 상승하는가
  5. 4단계 클리어까지의 총 플레이타임이 모바일 세션에 적합한가
